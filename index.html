☁️ Enable Cloud Sync (free):
1. Go to supabase.com → New project (free tier)
2. In SQL Editor, run:
create table wines (id text primary key, data jsonb, updated_at timestamptz default now()); alter table wines enable row level security; create policy "public" on wines for all using (true); 3. Copy your Project URL and anon key from Settings → API
4. Open wine-cellar-v2.html in a text editor, paste them into SUPABASE_URL and SUPABASE_KEY near the top of the <script> section

Once set up, all changes sync automatically and your cellar works on every device.
